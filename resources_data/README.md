# Resources Data — Documentacion Base

Este directorio contiene un proyecto de ejemplo base (`Control_ESP32C3_for_Web`) usado como referencia documental durante el desarrollo del robot autonomo ESP32-S3.

## Contenido

- `Control_ESP32C3_for_Web/` — Proyecto de referencia para ESP32-C3 con control web
- Los archivos aqui presentes son exclusivamente documentales y no forman parte del build del proyecto principal

## Nota

El proyecto principal se encuentra en `src/`, `include/` y `frontend/`.
